# gcp-data-product-template
Data Platform Owned Repository for Creating Data Products

To use this repo:
1. Navigate to Actions and click the `Create & Configure Repo` Action.
2. Click `Run workflow` and populate the input form according to your new product.
3. Run the workflow and this will create your new repo and configure it according to your inputs.
4. Navigate to the README.md file in your new data product repo to find instructions in how to deploy your data product
